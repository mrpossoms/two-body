ω

Is the argument of periapsis, meaning the angle from the line of the ascending node at which the satellite is closest to focus of the orbit. It is defined as `ω = cos^-1(n*e)` where `n` is the normalized line of the ascending node and `e` is the normalized eccentricity vector.

ν

Is the true anomaly. The true anomaly is the angle from the line of the ascending node at which the satellite was first observed. The true anomaly is defined as `ν = cos^-1(r_0*e)` where both `r_0` is the normalized position vector and `e` is the normalized eccentricity vector.

Ω

Is the longitude of the ascending node the angle from the geocentric basis vector `I` to the line of the ascending node.

e

Is the eccentricity and defines the shape of the orbit. 0 eccentricity means the orbit is perfectly circular. As the eccentricity approaches 1 the orbital path becomes more and more elliptical. The eccentricity vector is defined as
`e = 1/µ[(||v_0||^2 - µ/||r_0||)r_0 - (r_0 * v_0)v_0]`
where `r_0` and `v_0` are the position and velocity vectors respectively at the time of observation. `µ` is the gravitational parameter for the orbited body. In this case here, `µ` is set as 1 because we are using canonical units of time and space.

p

Is the semi-latus rectum, which is a line drawn perpendicular to the periapsis, and whose length extends from the focus, or central body of the orbit to the orbital path. The semi-latus rectum vector is defined as
`p = ||h||^2/µ`
where `h` is the angular momentum vector, and `µ` is the gravitational parameter for the orbited body. In this case here, `µ` is set as 1 because we are using canonical units of time and space.

i

is the inclination and defines the angle between the angular momentum vector and the basis vector K.
